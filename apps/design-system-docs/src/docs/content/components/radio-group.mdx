## Overview

Radio group lets users choose one option from a visible set.

## Usage

Use radio group when all options should stay visible and mutually exclusive. Prefer select when the option set is longer or when space is constrained.

## Style

Keep option labels comparable in length so users can scan the set quickly. Order options intentionally, not alphabetically by default.

## Code

```tsx
import { RadioGroup, RadioGroupItem } from "@limia/design-system";
```

## Accessibility

- Group radios under a clear shared label.
- Keep the selected state visible beyond color alone.
- Avoid splitting one decision across multiple radio groups.
