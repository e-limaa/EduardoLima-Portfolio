## Visão geral

Section é o wrapper de layout para superfícies editoriais do portfólio. Ele padroniza ritmo vertical, tratamento de fundo e comportamento opcional de container para que diferentes seções pareçam parte do mesmo sistema.

## Uso

Use Section para definir regiões principais da página, como blocos após o hero, listas de projetos, depoimentos ou áreas de contato. Mantenha lógica de negócio e narrativa específica dentro dos filhos em vez de adicionar props de layout para cada necessidade de produto.

## Estilo

`variant`, `grid`, `noise` e `container` permitem ajustar a atmosfera sem quebrar consistência. A recomendação padrão é `variant="subtle"` com `grid` e `noise` ativos em páginas editoriais com identidade de marca.

## Acessibilidade

- Section renderiza um elemento semântico `<section>` e normalmente deve conter um heading visível ou `aria-labelledby`.
- Camadas decorativas como grid e noise precisam continuar não essenciais para compreensão.
- Se a área for puramente visual e não representar um landmark útil, prefira um wrapper neutro em vez de adicionar landmarks desnecessários.
