## Overview

Switch toggles a setting on or off immediately.

## Usage

Use switch for binary settings that take effect instantly. Prefer checkbox when the control participates in a form submission rather than applying state immediately.

## Style

Pair switch with a direct label that reads as a stateful setting. If the consequence needs explanation, add helper text near the control.

## Code

```tsx
import { Switch } from "@limia/design-system";
```

## Accessibility

- Label the switch clearly with the setting being controlled.
- Use switch only when users can understand the on/off consequence immediately.
- Do not mix switch semantics with submit-based checkbox behavior.
