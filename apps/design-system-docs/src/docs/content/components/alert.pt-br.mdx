## Uso

- Use o alert padrao para mensagens informativas que precisam permanecer na tela.
- Use `variant="destructive"` para erros com consequencia ou urgencia.
- Prefira alert a toast quando a mensagem precisa continuar ligada ao formulario ou a secao da pagina.
