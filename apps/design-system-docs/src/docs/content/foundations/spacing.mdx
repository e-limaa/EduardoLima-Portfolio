## Overview

Spacing defines rhythm between elements, sections and interaction zones. A shared scale prevents drift and makes product layouts easier to scan, compare and maintain.

## Principles

Use spacing to express grouping and separation consistently. Tighter values imply relationship; larger values imply a shift in task, topic or hierarchy.

## Implementation

Use the shared scale for margin, padding, gap and layout scaffolding. Arbitrary values should be rare and justified by a real system constraint.

## Accessibility

Spacing affects readability, touch comfort and perceived grouping. Crowded layouts make scanning harder and reduce the clarity of focus and validation states.
