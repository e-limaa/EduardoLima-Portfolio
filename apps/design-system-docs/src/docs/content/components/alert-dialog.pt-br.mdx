## Visão geral

Alert dialog confirma ações destrutivas ou de alto impacto antes da execução.

## Uso

Use alert dialog quando a pessoa estiver prestes a disparar uma mudança destrutiva, custosa ou difícil de recuperar. Não use para confirmações rotineiras que só adicionam atrito sem aumentar segurança.

## Estilo

Mantenha o título direto e orientado à consequência. A pessoa precisa entender o que vai acontecer e se existe reversão antes de escolher.

## Código

```tsx
import {
  AlertDialog,
  AlertDialogAction,
  AlertDialogCancel,
  AlertDialogContent,
  AlertDialogDescription,
  AlertDialogTitle,
  AlertDialogTrigger,
} from "@limia/design-system";
```

## Acessibilidade

- O dialog mantém o foco preso enquanto estiver aberto.
- Título e descrição devem explicar consequência e recuperação com clareza.
- Ações destrutivas precisam de labels explícitos, como "Excluir projeto", em vez de genéricos como "Confirmar".
