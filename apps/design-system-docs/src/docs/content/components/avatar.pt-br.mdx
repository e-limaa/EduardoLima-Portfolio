## Visão geral

Avatar exibe mídia de identidade para pessoas, times e bots em uma superfície compacta e reutilizável.

## Uso

Use avatar quando identidade ajudar a pessoa a escanear conversas, atribuições, ownership ou autoria. Combine com labels de texto em contextos densos em que iniciais sozinhas não bastam.

## Estilo

Mantenha o tamanho dos avatares consistente dentro do mesmo grupo e trate o fallback como um estado desenhado e suportado, não como erro.

## Código

```tsx
import { Avatar, AvatarFallback, AvatarImage } from "@limia/design-system";
```

## Acessibilidade

- Forneça `alt` significativo quando a imagem acrescentar informação de identidade.
- Use fallback estável para que layout e reconhecimento sobrevivam a falhas de carregamento.
- Não dependa apenas do avatar para comunicar ownership ou estado.
