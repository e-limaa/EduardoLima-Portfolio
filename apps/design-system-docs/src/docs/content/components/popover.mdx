## Overview

Popover reveals contextual content in a lightweight floating surface.

## Usage

Use popover when content is user-invoked, contextual and small enough to keep the current page as the main frame of reference. Avoid it for long forms, complex decisions or critical flows that deserve a dialog.

## Style

Keep the title, copy and actions compact. Popover should help users act quickly without creating a second full-screen mental model.

## Code

```tsx
import { Popover, PopoverContent, PopoverTrigger } from "@limia/design-system";
```

## Accessibility

- Triggers must be clearly labeled.
- Focus should move predictably into the popover when interaction begins.
- Do not place long-form critical content in a popover.
