## Critérios de Promoção

Para promover de **Beta** para **Stable**:

- [ ] **Acessibilidade**: Passar em testes WCAG AA e navegação por teclado.
- [ ] **Cobertura**: Testes unitários > 80%.
- [ ] **Documentação**: Página MDX completa com exemplos.
- [ ] **Uso**: Usado com sucesso em produção.
