## Overview

Select helps users choose a single option from a structured list while keeping the interface compact.

## Usage

Use select for concise single-choice inputs with more than a few options. Prefer radio group when all options should remain visible at once.

## Style

Keep placeholder text short and labels external to the trigger when the field needs more context. Group options only when the grouping helps scanning.

## Code

```tsx
import {
  Select,
  SelectContent,
  SelectItem,
  SelectTrigger,
  SelectValue,
} from "@limia/design-system";
```

## Accessibility

- The field still needs a visible label in most forms.
- Option labels should be concise and unambiguous.
- Avoid using select for binary decisions better served by switch or radio.
