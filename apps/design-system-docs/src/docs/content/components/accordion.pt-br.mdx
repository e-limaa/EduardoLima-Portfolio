## Visão geral

Accordion revela conteúdo denso de forma progressiva, mantendo seções relacionadas dentro do mesmo fluxo.

## Uso

Use accordion quando o conteúdo for relacionado, mas não tiver a mesma prioridade no primeiro olhar.

- Use `type="single"` quando apenas um painel deve permanecer aberto por vez.
- Use `type="multiple"` para filtros, configurações ou fluxos de inspeção.
- Mantenha os labels dos triggers curtos o bastante para leitura em lista.

## Estilo

Os triggers devem se comportar como labels concisos de seção, não como resumos longos. Mantenha os blocos internos compactos e evite aninhar interações complexas em muitos níveis de disclosure.

## Código

```tsx
import {
  Accordion,
  AccordionContent,
  AccordionItem,
  AccordionTrigger,
} from "@limia/design-system";
```

## Acessibilidade

- Triggers são acessíveis por teclado por padrão.
- Use labels claros para que o conteúdo recolhido continue compreensível no estado fechado.
- Não esconda validação crítica, informação legal ou consequências destrutivas dentro de painéis colapsados.
