## Overview

Toggle group organizes related toggles into a structured pressed-state set.

## Usage

Use toggle group when multiple related pressed states must be presented together. Choose `type="single"` for mutually exclusive states and `type="multiple"` for combinable options.

## Style

Option labels should be comparable and visually balanced. Large differences in label length make the group harder to scan and interpret.

## Code

```tsx
import { ToggleGroup, ToggleGroupItem } from "@limia/design-system";
```

## Accessibility

- Give the group a clear surrounding label in forms or toolbars.
- Use icon-only items only when each one has an accessible label.
- Keep the selection model obvious: single choice should not look like multi-select.
