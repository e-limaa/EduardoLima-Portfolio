## Visão geral

Popover revela conteúdo contextual em uma superfície flutuante leve.

## Uso

Use popover quando o conteúdo for disparado pela pessoa, contextual e pequeno o bastante para manter a página atual como referência principal. Evite para formulários longos, decisões complexas ou fluxos críticos que mereçam um dialog.

## Estilo

Mantenha título, texto e ações compactos. Popover deve ajudar a agir rápido sem criar um segundo modelo mental de tela inteira.

## Código

```tsx
import { Popover, PopoverContent, PopoverTrigger } from "@limia/design-system";
```

## Acessibilidade

- Triggers precisam de labels claros.
- O foco deve entrar no popover de forma previsível quando a interação começar.
- Não coloque conteúdo crítico e extenso dentro de popover.
