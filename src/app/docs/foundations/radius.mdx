
# Radius

Border radius tokens smooth the corners of UI elements. Our system uses a smart base radius that scales automatically across components.

<Callout type="info" title="Adaptive Radius">
  Our radius system is built on variables. Changing `--radius` in `globals.css` updates the entire system (buttons, inputs, cards) instantly.
</Callout>

## Usage Guidelines

`rounded-md` is typically used for small interactive elements like Buttons and Inputs inside cards.
`rounded-lg` or `rounded-xl` is used for containers like Cards, Dialogs, and Popovers.

<DoDontContainer>
  <DoDont type="do" title="Nest Radii Correctly">
    Inner elements should generally have less radius than their containers to look visually concentric.
  </DoDont>
  <DoDont type="dont" title="Mix Shapes">
    Avoid mixing fully rounded (pill) buttons with sharp rectangular inputs in the same group.
  </DoDont>
</DoDontContainer>
