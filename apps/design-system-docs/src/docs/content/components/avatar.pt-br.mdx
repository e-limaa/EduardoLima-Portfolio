## Uso

Use Avatar quando uma pessoa, assistente ou organização precisa de identidade visual reconhecível em pouco espaço.

## Anatomia

```tsx
import { Avatar, AvatarFallback, AvatarImage } from '@limia/design-system';

export function Example() {
  return (
    <Avatar>
      <AvatarImage src="/avatar.png" alt="Eduardo Lima" />
      <AvatarFallback>EL</AvatarFallback>
    </Avatar>
  );
}
```

## Diretrizes

- Prefira fallback curto, como iniciais ou rótulos de assistente.
- Sempre passe `alt` significativo quando a imagem comunicar identidade.
- Trate o Avatar como decorativo apenas quando o texto ao redor já nomear o sujeito.
