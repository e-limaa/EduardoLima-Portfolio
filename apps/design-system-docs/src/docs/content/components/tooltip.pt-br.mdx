## Visão geral

Tooltip fornece orientação contextual curta em hover ou foco.

## Uso

Use tooltip para hints de apoio concisos, como esclarecimento de ícones ou labels compactos. Não use para instruções obrigatórias, explicações complexas ou feedback crítico de status.

## Estilo

Mantenha o texto do tooltip em uma frase curta. Se o conteúdo exigir formatação, decisões ou tempo maior de leitura, use popover ou outro pattern mais rico.

## Código

```tsx
import {
  Tooltip,
  TooltipContent,
  TooltipProvider,
  TooltipTrigger,
} from "@limia/design-system";
```

## Acessibilidade

- O conteúdo do tooltip também precisa aparecer em foco, não apenas em hover.
- Não coloque informação essencial apenas dentro do tooltip.
- Os triggers devem continuar fazendo sentido sem o tooltip.
