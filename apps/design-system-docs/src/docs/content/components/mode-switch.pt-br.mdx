## Visão geral

Mode Switch reúne os switches binários estilizados construídos sobre o primitive base `Switch`. Use quando o próprio controle precisa comunicar diretamente o modo atual, seja com ícones ou com labels curtos em texto.

## Uso

Use `ThemeSwitch` para alternância de tema, como light e dark. Use `TextSwitch` para labels binários curtos, como idioma ou modo compacto, quando as duas opções precisam permanecer visíveis dentro do controle.

## Estilo

Essas variantes preservam a mesma geometria do padrão de switch binário: trilho neutro, um thumb móvel e dois slots simétricos. `ThemeSwitch` usa a superfície primária para reforçar a intenção de light versus dark, enquanto `TextSwitch` mantém o thumb selecionado mais neutro para não competir com a navegação ao redor.

<Callout type="info" title="Use apenas labels curtos">
  `TextSwitch` funciona melhor com labels de duas ou três letras, como `EN/PT`, `ON/OFF` ou `A/B`. Textos maiores quebram o ritmo visual e devem migrar para tabs ou segmented controls.
</Callout>

## Código

Use as variantes estilizadas direto de `@limia/design-system` quando o switch base não for expressivo o suficiente sozinho.

```tsx
import { ThemeSwitch, TextSwitch } from "@limia/design-system";

export function Example() {
  return (
    <div className="flex items-center gap-4">
      <ThemeSwitch
        checked={theme === "dark"}
        onCheckedChange={(checked) => setTheme(checked ? "dark" : "light")}
        aria-label={theme === "dark" ? "Alternar para tema claro" : "Alternar para tema escuro"}
      />

      <TextSwitch
        checked={language === "pt-br"}
        onCheckedChange={(checked) => setLanguage(checked ? "pt-br" : "en")}
        uncheckedLabel="EN"
        checkedLabel="PT"
        aria-label="Selecionar idioma"
      />
    </div>
  );
}
```

## Acessibilidade

- Forneça um `aria-label` que descreva a ação ou o significado atual do switch.
- Mantenha labels curtos e sem ambiguidade para que cada lado continue legível dentro do controle.
- Use essas variantes apenas para mudanças imediatas de modo, não para escolhas enviadas em formulário.
