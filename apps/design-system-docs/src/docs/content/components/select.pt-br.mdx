## Visão geral

Select ajuda a escolher uma única opção em uma lista estruturada mantendo a interface compacta.

## Uso

Use select para escolhas únicas e concisas quando houver mais do que poucas opções. Prefira radio group quando todas as opções precisarem ficar visíveis ao mesmo tempo.

## Estilo

Mantenha o placeholder curto e o label fora do trigger quando o campo exigir mais contexto. Agrupe opções apenas quando isso realmente ajudar no scan.

## Código

```tsx
import {
  Select,
  SelectContent,
  SelectItem,
  SelectTrigger,
  SelectValue,
} from "@limia/design-system";
```

## Acessibilidade

- O campo continua precisando de label visível na maioria dos formulários.
- Labels das opções devem ser concisos e sem ambiguidade.
- Evite usar select para decisões binárias que funcionam melhor com switch ou radio.
