## Visão geral

Sheet abre fluxos de apoio a partir da borda da viewport, preservando o contexto da página.

## Uso

Use sheet para edição secundária, configurações de apoio e tarefas contextuais que se beneficiam de continuar ancoradas à página atual. Prefira dialog quando a tarefa for mais crítica do que a página de fundo.

## Estilo

Mantenha o conteúdo do sheet focado e organizado na vertical. A entrada pela borda deve parecer uma extensão do fluxo atual, não uma página escondida.

## Código

```tsx
import {
  Sheet,
  SheetContent,
  SheetHeader,
  SheetTitle,
  SheetTrigger,
} from "@limia/design-system";
```

## Acessibilidade

- Sheet gerencia o foco enquanto estiver aberto.
- Título e descrição devem explicar para que o painel serve antes que a pessoa alcance os campos.
- Evite sobrecarregar o sheet com fluxos longos e multi-step.
