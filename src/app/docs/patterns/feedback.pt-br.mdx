## Princípios

O sistema deve sempre manter os usuários informados sobre o que está acontecendo, através de feedback apropriado em um tempo razoável.

## Estados de Carregamento

Use spinners de carregamento dentro de botões para ações iniciadas pelo usuário.

<ComponentPreview>
  <div className="flex gap-4">
    <Button disabled>
        <span className="mr-2 animate-spin">⟳</span>
        Salvando...
    </Button>
    <Button variant="secondary" disabled>
        Processando
    </Button>
  </div>
</ComponentPreview>
