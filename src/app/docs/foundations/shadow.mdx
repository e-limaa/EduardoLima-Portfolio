
# Shadows & Elevation

Elevation serves to lift elements from the background, indicating hierarchy and interactivity. We use shadows to simulate this elevation Z-axis.

## Usage

- **shadow-sm**: Subtle borders, interactive states like hover on table rows.
- **shadow (default)**: Cards, dropdowns, low-level poppers.
- **shadow-lg**: Modals, Dialogs, Floating Action Buttons.
