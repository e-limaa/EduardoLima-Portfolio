Every component in the Design System moves through defined stages of maturity. This helps consumers understand the stability and support level of the tools they are using.

## Promotion Criteria

To promote a component from **Beta** to **Stable**, it must meet the following criteria:

- [ ] **Accessibility**: Passes automated WCAG AA tests and basic keyboard navigation manual tests.
- [ ] **Coverage**: Has at least 80% unit test coverage.
- [ ] **Documentation**: Has a complete MDX page with examples and props table.
- [ ] **Usage**: Is used successfully in at least one production application.
