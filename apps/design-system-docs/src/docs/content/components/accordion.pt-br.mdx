## Uso

- Use `type="single"` quando apenas um painel deve permanecer aberto.
- Use `type="multiple"` em filtros, configuracoes ou fluxos de inspecao.
- Mantenha labels claras no trigger e conteudo objetivo.

## Acessibilidade

- Os botões do trigger sao acessiveis por teclado por padrao.
- Nao esconda validacoes obrigatorias, precificacao ou consequencias destrutivas em conteudo recolhido.
