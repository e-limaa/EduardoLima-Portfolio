## Usage

Use Avatar when a person, assistant or organization needs a recognizable visual identity in a compact space.

## Anatomy

```tsx
import { Avatar, AvatarFallback, AvatarImage } from '@limia/design-system';

export function Example() {
  return (
    <Avatar>
      <AvatarImage src="/avatar.png" alt="Eduardo Lima" />
      <AvatarFallback>EL</AvatarFallback>
    </Avatar>
  );
}
```

## Guidance

- Prefer short fallback content such as initials or assistant labels.
- Always pass meaningful `alt` text when an image conveys identity.
- Keep Avatar decorative only when the surrounding text already names the subject.
