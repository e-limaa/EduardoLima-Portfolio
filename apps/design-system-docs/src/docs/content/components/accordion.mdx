## Usage

- Use `type="single"` when only one panel should stay open.
- Use `type="multiple"` for filters, settings or inspection flows.
- Keep trigger labels clear and content concise.

## Accessibility

- Accordion trigger buttons are keyboard accessible by default.
- Do not hide required validation, pricing or destructive consequences inside collapsed content.
