## Visão geral

Toggle group organiza toggles relacionados em um conjunto estruturado de estados pressionados.

## Uso

Use toggle group quando múltiplos estados pressionados relacionados precisarem ser apresentados juntos. Escolha `type="single"` para estados mutuamente exclusivos e `type="multiple"` para opções combináveis.

## Estilo

Os labels das opções devem ser comparáveis e visualmente equilibrados. Diferenças grandes de comprimento dificultam scan e interpretação.

## Código

```tsx
import { ToggleGroup, ToggleGroupItem } from "@limia/design-system";
```

## Acessibilidade

- Dê ao grupo um label de contexto claro em formulários ou toolbars.
- Use itens só com ícone apenas quando cada um tiver label acessível.
- Mantenha o modelo de seleção óbvio: escolha única não deve parecer multi-select.
