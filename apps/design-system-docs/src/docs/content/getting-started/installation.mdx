## Overview

Install Limia as a managed dependency. Consumers should import the system packages directly instead of copying primitives or recreating the theme locally.

## Setup

### Requirements

- Node.js 18+
- React 18+
- Tailwind CSS 3.3+

### Install the packages

```bash
npm install @limia/design-system @limia/tokens
```

### Import styles

```tsx
import "@limia/design-system/styles.css";
import "@limia/tokens/css";
```

### Configure Tailwind

```js
import { limiaPreset } from "@limia/design-system/tailwind";

export default {
  content: [
    "./src/**/*.{ts,tsx}",
    "./node_modules/@limia/design-system/dist/**/*.{js,mjs}"
  ],
  presets: [limiaPreset],
  darkMode: ["class"],
};
```

## Usage

Once styles and preset wiring are in place, primitives are ready to use without additional setup.

<ComponentPreview>
  <div className="flex items-center justify-center gap-4 p-8">
    <Button>Hello World</Button>
    <Button variant="secondary">Secondary action</Button>
  </div>
</ComponentPreview>

```tsx
import { Button } from "@limia/design-system";

export default function App() {
  return (
    <div className="flex gap-4">
      <Button>Hello World</Button>
      <Button variant="secondary">Secondary action</Button>
    </div>
  );
}
```
