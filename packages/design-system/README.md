# Antigravity Design System

O Design System oficial para o portfólio de Eduardo Lima. Tratado como um produto independente, este sistema foca em uma estética premium ("Zinc/Slate") com suporte nativo a temas escuro e claro.

## Princípios
- **Premium Aesthetics**: Cores curadas, grid sutil e micro-animações.
- **Accessibility**: Baseado em Radix UI.
- **Performance**: Estilizado com Tailwind CSS v4.

## Estrutura
- `/src/components/ui`: Componentes atômicos e moleculares.
- `/src/tokens`: Definições de cores, tipografia e espaçamento.
