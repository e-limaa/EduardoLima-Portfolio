## Visão geral

Radio group permite escolher uma opção em um conjunto visível.

## Uso

Use radio group quando todas as opções precisarem permanecer visíveis e mutuamente exclusivas. Prefira select quando o conjunto for maior ou quando o espaço estiver mais restrito.

## Estilo

Mantenha labels de opção comparáveis em comprimento para facilitar o scan. Ordene as opções com intenção, e não alfabeticamente por padrão.

## Código

```tsx
import { RadioGroup, RadioGroupItem } from "@limia/design-system";
```

## Acessibilidade

- Agrupe os radios sob um label compartilhado e claro.
- Mantenha o estado selecionado visível além da cor.
- Evite dividir uma única decisão em múltiplos radio groups.
