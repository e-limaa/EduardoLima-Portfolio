## Visão geral

Separator divide regiões relacionadas de conteúdo sem adicionar ruído visual.

## Uso

Use separator quando o agrupamento sozinho não bastar para distinguir regiões adjacentes. Evite usá-lo como substituto de espaçamento ou de hierarquia por heading.

## Estilo

Separators devem ser sutis e secundários. Se a divisão parecer forte demais, provavelmente o layout precisa de ritmo, não de mais linhas.

## Código

```tsx
import { Separator } from "@limia/design-system";
```

## Acessibilidade

- Use a orientação correta para separadores verticais e horizontais.
- Não dependa do separator como único sinal de hierarquia.
- Mantenha títulos e labels de seção disponíveis para tecnologia assistiva.
