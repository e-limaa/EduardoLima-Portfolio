## Uso

- Use radio group quando o usuario precisa escolher exatamente uma opcao em um conjunto curto e visivel.
- Prefira select quando a lista for maior ou o espaco em tela estiver limitado.
- Mantenha labels mutuamente exclusivas e faceis de comparar.
