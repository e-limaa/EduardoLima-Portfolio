
# Spacing

Our spacing scale is the foundation of vertical and horizontal rhythm. It ensures consistency in margins, padding, sizing, and layout.

## Scale

We use the standard Tailwind spacing scale.

## Usage Guidelines

<DoDontContainer>
  <DoDont type="do" title="Use the scale">
    Always use standard values like `p-4`, `m-6`, `gap-2`.
  </DoDont>
  <DoDont type="dont" title="Magic Numbers">
     Avoid arbitrary values like `margin: 19px` or `w-[123px]` unless absolutely necessary for pixel-perfect overlays.
  </DoDont>
</DoDontContainer>
