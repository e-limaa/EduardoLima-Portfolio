## Usage

- Use checkbox for independent selections.
- Use radio group or select when users must choose only one option.
- Keep labels explicit enough to stand without surrounding context.

## Accessibility

- Always associate the control with a visible label.
- Expand the hit area by wrapping checkbox and text in a shared row.
