## Visão geral

Table exibe dados estruturados em linhas e colunas com hierarquia clara.

## Uso

Use table quando a pessoa precisar escanear, comparar ou inspecionar conjuntos estruturados de dados. Evite em telas pequenas quando card ou lista comunicarem a mesma informação com mais clareza.

## Estilo

Mantenha labels de coluna curtos, alinhe valores de modo consistente e evite sobrecarregar as linhas com ações sem relação. Tabelas funcionam melhor quando a hierarquia fica clara antes da interação.

## Código

```tsx
import {
  Table,
  TableBody,
  TableCell,
  TableHead,
  TableHeader,
  TableRow,
} from "@limia/design-system";
```

## Acessibilidade

- Use células de cabeçalho para contexto de coluna e linha quando apropriado.
- Não dependa apenas de alinhamento ou cor para comunicar significado.
- Forneça alternativas responsivas quando tabelas densas quebrarem no mobile.
