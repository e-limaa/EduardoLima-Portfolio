## Overview

Checkbox lets users select one or more independent options.

## Usage

Use checkbox when options are independent and can be combined. Prefer radio group when the user must choose exactly one option from a visible set.

## Style

Keep labels close to the control and use helper text only when the choice needs extra explanation. Avoid long, dense paragraphs as checkbox labels.

## Code

```tsx
import { Checkbox, Label } from "@limia/design-system";
```

## Accessibility

- Associate each checkbox with a visible label.
- Group related checkboxes with a clear heading or fieldset.
- Do not hide the consequence of a checked state in distant copy.
