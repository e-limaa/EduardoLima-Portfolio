## Visão geral

Dropdown menu expõe ações contextuais compactas e seleção leve em uma superfície flutuante.

## Uso

Use dropdown menu para ações locais ligadas ao objeto ou região atual. Evite usá-lo como substituto de navegação principal ou para listas longas e complexas de opções.

## Estilo

Mantenha os labels concisos, agrupe ações quando necessário e separe ações destrutivas das rotineiras. O menu deve permitir leitura em uma passada.

## Código

```tsx
import {
  DropdownMenu,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuTrigger,
} from "@limia/design-system";
```

## Acessibilidade

- Menus são navegáveis por teclado por padrão.
- O label do trigger deve explicar que tipo de menu está sendo aberto.
- Não esconda caminhos críticos de navegação dentro de menus contextuais.
