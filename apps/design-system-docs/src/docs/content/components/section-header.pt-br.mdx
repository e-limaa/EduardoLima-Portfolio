## Visão geral

Section Header é o bloco padronizado de introdução para regiões principais da página. Ele mantém heading, texto de apoio e metadados opcionais visualmente alinhados entre diferentes seções editoriais.

## Uso

Use Section Header no topo de seções longas como Projetos, Serviços ou Depoimentos. Ele é um componente de composição, então faz mais sentido quando a página precisa repetir um padrão de abertura, e não como um heading genérico.

## Estilo

`align` muda o layout do texto, enquanto `index` e `label` adicionam metadados editoriais. Use `children` apenas quando você realmente precisar de uma ação ou badge customizado no lugar do slot padrão.

## Acessibilidade

- Use `headingId` quando o wrapper ou corpo da seção precisar de `aria-labelledby`.
- Mantenha o título descritivo o suficiente para funcionar sozinho na navegação por headings do leitor de tela.
- Se `children` adicionar controles interativos, garanta que eles continuem alcançáveis por teclado e não substituam o conteúdo essencial do heading.
