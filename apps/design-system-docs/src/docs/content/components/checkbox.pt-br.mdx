## Visão geral

Checkbox permite selecionar uma ou mais opções independentes.

## Uso

Use checkbox quando as opções forem independentes e puderem ser combinadas. Prefira radio group quando a pessoa precisar escolher exatamente uma opção em um conjunto visível.

## Estilo

Mantenha os labels próximos do controle e use helper text apenas quando a escolha exigir contexto extra. Evite transformar o label em um bloco longo e denso de texto.

## Código

```tsx
import { Checkbox, Label } from "@limia/design-system";
```

## Acessibilidade

- Associe cada checkbox a um label visível.
- Agrupe checkboxes relacionados com heading claro ou fieldset.
- Não esconda a consequência do estado marcado em um texto distante.
