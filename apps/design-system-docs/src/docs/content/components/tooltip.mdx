## Overview

Tooltip provides short contextual guidance on hover or focus.

## Usage

Use tooltip for concise supporting hints, such as icon clarification or compact labels. Do not use it for required instructions, complex explanations or critical status feedback.

## Style

Keep tooltip copy to a short phrase or sentence. If the content needs formatting, decisions or longer reading time, use popover or another richer pattern.

## Code

```tsx
import {
  Tooltip,
  TooltipContent,
  TooltipProvider,
  TooltipTrigger,
} from "@limia/design-system";
```

## Accessibility

- Tooltip content must also be available on focus, not hover only.
- Do not place essential information only in a tooltip.
- Triggers should still make sense without the tooltip.
