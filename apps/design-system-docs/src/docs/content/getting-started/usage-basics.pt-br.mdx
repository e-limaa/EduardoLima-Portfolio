## Visão geral

Use o Limia como sistema, não como um conjunto solto de peças. A adoção só está correta quando os apps consomem exports públicos, tokens semânticos e padrões documentados sem recriar uma camada local de UI.

## Setup

Comece pelos entrypoints do pacote e pelos estilos compartilhados:

- `@limia/design-system` para primitives e patterns
- `@limia/tokens` para CSS de tokens e referências de token
- o preset compartilhado de Tailwind quando o app host usa Tailwind

## Uso

Adote o sistema nesta ordem:

1. Substitua primitives locais pelos exports públicos do Limia.
2. Remova cores e espaçamentos hardcoded em favor de tokens semânticos.
3. Mantenha composição específica de produto na camada do app, não dentro do pacote do design system.
