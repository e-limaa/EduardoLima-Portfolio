import { Button } from '@limia/design-system';

## Visão geral

Instale o Limia como uma dependência gerenciada. Apps consumidores devem importar os pacotes do sistema diretamente, em vez de copiar primitives ou recriar o tema localmente.

## Setup

### Requisitos

- Node.js 18+
- React 18+
- Tailwind CSS 3.3+

### Instale os pacotes

```bash
npm install @limia/design-system @limia/tokens
```

### Importe os estilos

```tsx
import "@limia/design-system/styles.css";
import "@limia/tokens/css";
```

### Configure o Tailwind

```js
import { limiaPreset } from "@limia/design-system/tailwind";

export default {
  content: [
    "./src/**/*.{ts,tsx}",
    "./node_modules/@limia/design-system/dist/**/*.{js,mjs}"
  ],
  presets: [limiaPreset],
  darkMode: ["class"],
};
```

## Uso

Depois que estilos e preset estiverem ligados, os primitives ficam prontos para uso sem configuração adicional.

<ComponentPreview>
  <div className="flex items-center justify-center gap-4 p-8">
    <Button>Olá mundo</Button>
    <Button variant="secondary">Ação secundária</Button>
  </div>
</ComponentPreview>

```tsx
import { Button } from "@limia/design-system";

export default function App() {
  return (
    <div className="flex gap-4">
      <Button>Olá mundo</Button>
      <Button variant="secondary">Ação secundária</Button>
    </div>
  );
}
```
