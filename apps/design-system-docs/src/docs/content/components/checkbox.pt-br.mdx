## Uso

- Use checkbox para selecoes independentes.
- Use radio group ou select quando o usuario precisar escolher apenas uma opcao.
- Mantenha labels explicitas o suficiente para funcionarem sem contexto extra.

## Acessibilidade

- Sempre associe o controle a um rotulo visivel.
- Aumente a area de clique agrupando checkbox e texto na mesma linha.
