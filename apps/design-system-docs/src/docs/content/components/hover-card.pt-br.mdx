## Visão geral

Hover card revela uma prévia contextual leve em hover ou foco.

## Uso

Use hover card para metadados suplementares, como prévias de perfil, labels ou pequenos detalhes de referência. Não use para informação crítica necessária para completar a tarefa.

## Estilo

Mantenha o conteúdo curto e fácil de escanear. Hover card deve parecer uma prévia, não um dialog completo ou um formulário.

## Código

```tsx
import {
  HoverCard,
  HoverCardContent,
  HoverCardTrigger,
} from "@limia/design-system";
```

## Acessibilidade

- O conteúdo também precisa ser alcançável por foco, e não apenas por hover.
- Não coloque ações obrigatórias ou instruções essenciais apenas dentro do card.
- O texto do trigger deve continuar significativo fora da prévia.
