## Overview

Avatar displays identity media for people, teams and bots in a compact, reusable surface.

## Usage

Use avatar when identity helps users scan conversations, assignments, ownership or authorship. Pair it with text labels in dense contexts where initials alone are not enough.

## Style

Keep avatar sizing consistent inside the same cluster and use the fallback state as a designed, supported state rather than as an error condition.

## Code

```tsx
import { Avatar, AvatarFallback, AvatarImage } from "@limia/design-system";
```

## Accessibility

- Provide meaningful `alt` text when the image adds identity information.
- Use a stable fallback so the layout and recognition survive failed image loading.
- Do not rely on avatar alone to communicate ownership or state.
