## Diretrizes de Uso

### Acessibilidade

Garantimos taxas de contraste WCAG AA para todos os pares de texto padrão.

<Callout type="warning" title="Checagem de Contraste">
  Nunca confie apenas na cor para transmitir significado. Sempre use rótulos de texto ou ícones junto com cores de status para apoiar usuários daltônicos.
</Callout>

### O Que Fazer

<DoDontContainer>
  <DoDont type="do" title="Use Nomes Semânticos">
    Use `bg-destructive` para erros ou ações perigosas. Isso lida automaticamente com a escala de vermelho e modo escuro.
  </DoDont>
  <DoDont type="dont" title="Hardcode de Hex">
    Evite usar valores hex arbitrários como `#ff0000`. Isso quebra o tema e requer manutenção manual.
  </DoDont>
</DoDontContainer>
