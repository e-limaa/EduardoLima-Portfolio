## Overview

Accordion reveals dense content progressively while keeping related sections grouped in the same flow.

## Usage

Use accordion when content is related but not equally important at first glance.

- Use `type="single"` when one panel should stay open at a time.
- Use `type="multiple"` for filters, settings or inspection workflows.
- Keep trigger labels short enough to scan as a list.

## Style

Triggers should read as concise section labels, not paragraph summaries. Keep content blocks compact and avoid nesting complex interaction inside many levels of disclosure.

## Code

```tsx
import {
  Accordion,
  AccordionContent,
  AccordionItem,
  AccordionTrigger,
} from "@limia/design-system";
```

## Accessibility

- Triggers are keyboard accessible by default.
- Use clear labels so collapsed content remains understandable in the closed state.
- Do not hide critical validation, legal or destructive information inside collapsed panels.
