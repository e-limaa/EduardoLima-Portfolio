Acessibilidade não é uma feature, é um requisito. Nosso Design System segue as diretrizes da WCAG 2.1 nível AA.

## Princípios (POUR)

## Teclado

Todos os componentes interativos devem ser acessíveis via teclado.

- **Tab**: Navega para o próximo elemento focado.
- **Enter / Space**: Ativa o elemento.
- **Esc**: Fecha modais ou popovers.
- **Setas**: Navega entre itens de lista (menus, radio groups).
