## Visão geral

Switch alterna uma configuração entre ligado e desligado imediatamente.

## Uso

Use switch para configurações binárias que produzem efeito imediato. Prefira checkbox quando o controle participar de um envio de formulário em vez de aplicar estado instantaneamente.

## Estilo

Combine o switch com um label direto que leia como configuração com estado. Se a consequência precisar de explicação, adicione helper text próximo ao controle.

## Código

```tsx
import { Switch } from "@limia/design-system";
```

## Acessibilidade

- Dê ao switch um label claro com a configuração que ele controla.
- Use switch apenas quando a consequência de ligado/desligado for imediatamente compreensível.
- Não misture semântica de switch com comportamento de checkbox baseado em submit.
